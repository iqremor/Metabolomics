Bienvenido al uso de la Herramienta de Scrapping
---------------------------------------------------------------------
La ejecución del programa se debe realizar siempre desde la CONSOLA 
para evitar cualquier problema.
1-Antes de iniciar el programa debe revisar que NO exista la carpeta resultados
de existir esta carpeta cambiele el nombre o muevala a otro repositorio.
2-La base de datos debe ser en la extensión tipo (.xlsx) y contar con las columnas de nombre ["PubChem","origen","SMILES"]
3-el script de scrapping funciona solo con el navegador mozilla
